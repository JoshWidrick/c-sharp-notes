C# (C Sharp) Language Notes
===
This is a collection of C# language notes covering everything from the language basics, to more advanced features. All of the note sections are in the form of individual C# projects. It is recommended that these notes are viewed from within [Microsoft's Visual Studio](https://visualstudio.microsoft.com/).
